# Simple Interest Calculator

This project is a **Simple Interest Calculator** that calculates the interest based on principal, rate of interest, and time period.

## Features
- Calculate simple interest using the formula: `SI = (P * R * T) / 100`
- Input principal amount, rate of interest, and time period
- Output the calculated interest
